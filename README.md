# DaliScript 💀

A Fabric mod for Minecraft 26.1.2 (Tiny Takeover)

## Commands
- `/dali` — Check if mod is active
- `/gm <0-3>` — Change game mode
- `/ai <question>` — Simple AI assistant

## Info
- Minecraft: 26.1.2
- Loader: Fabric
- Versions: 35 development versions
